## About
**Greeting Website Loader** is a minimal, dependency-free preloader animation for websites and single-page apps.  
It displays greetings in multiple languages on a pure black backdrop with smooth fade and subtle scale transitions — ideal as an elegant first impression while your app loads.

**What it does**
- Cycles through greetings: `Hello`, `नमस्ते`, `Hallo`, `Bonjour`, `Hola`, `こんにちは`, `안녕하세요`, `Ciao`, `Olá`, `Привет`, `你好`
- Uses simple CSS transitions for fade + subtle scale
- Single-file React component (or plain HTML/CSS alternative)
